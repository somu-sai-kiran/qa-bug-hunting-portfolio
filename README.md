# QA Bug Hunting Portfolio

This repository contains real-world software testing investigations performed on publicly accessible applications.

The objective is to practice and demonstrate:

* Exploratory Testing
* Defect Investigation
* API Validation
* Browser DevTools Analysis
* Cross-Browser Testing
* Root Cause Analysis
* Professional Bug Reporting

Each case study includes:

* Issue Summary
* Investigation Process
* Evidence Collection
* Business Impact Analysis
* Findings and Conclusions

The purpose of this repository is educational and professional development. All observations were identified during normal application usage without performing any intrusive or unauthorized testing.
